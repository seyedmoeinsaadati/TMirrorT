TMirrorT unity package (version 1.0.0)
@copyright 2021, Seyed Moein Saadati, All rights reserved.

features:
- redesign mirror system
- Create Axis Mirror that allow you mirror object to x, y, z axis.
- Create Plane Mirror that allow you mirror object to xz, xy, yz plane.
- Create Point Mirror that allow you mirror object to a point with custom rotation axes. (you can set up your custom mirror)
- MirrorMaterial: mirror material object
